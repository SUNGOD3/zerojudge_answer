# k680 - 肥余歷險記---(反數學城2)

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=k680)

## 題目描述

題目要求讀取一組算式，每個算式包含一個運算符（+ 或 -）和兩個整數，然後計算算式結果並輸出。輸入的第一行是測資數量，後續每一行代表一個算式。

## 解題思路

這題的解題思路非常簡單，直接根據輸入的運算符進行加法或減法運算即可。程式碼首先讀取測資數量，然後迴圈讀取每個算式，判斷運算符是加法還是減法，並計算結果後輸出。

## 複雜度分析
- 時間複雜度: O(n)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int n,x,y,z;
char c;
int main(){
	cin >> n;
	for(int i=1;i<=n;++i){
		cin >> c >> x >> c >> y;
		if(c=='-')cout << x-y << "\n";
		else cout << x+y << "\n";
	}
}
