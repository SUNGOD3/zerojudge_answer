# f925 - Is CIA Today?

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=f925)

## 題目描述
題目要求判斷使用者輸入的月份和日期是否為 6 月 12 日。如果是，則輸出 "Yes"，否則輸出 "No"。

## 解題思路
此題為一個簡單的條件判斷問題。程式需要讀取兩個整數，分別代表月份和日期，然後檢查它們是否等於 6 和 12。如果兩者都相等，則輸出 "Yes"，否則輸出 "No"。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int main(){
	int a,b;
	cin >> a >> b;
	if(a==6&&b==12)cout << "Yes";
	else cout << "No";
}
