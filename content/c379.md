# c379 - 成為出題者

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=c379)

## 題目描述
題目要求計算至少需要答對多少題才能成為 ZeroJudge 的出題者。出題者的條件是答對至少 30% 的題目。輸入為題目總數 x，輸出為至少需要答對的題目數。

## 解題思路
題目非常簡單，只需要將輸入的題目總數 x 乘以 0.3 即可得到至少需要答對的題目數。由於題目保證 x 是 10 的倍數，因此計算結果會是整數。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>

using namespace std;

int main (){
	
	long long int a=0;
	
	while(cin >> a){
		cout << a*0.3 << endl; 
	}
	
}
