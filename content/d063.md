# d063 - 0 與 1

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d063)

## 題目描述
題目要求讀取一個整數，如果輸入為 0，則輸出 1；如果輸入為 1，則輸出 0。

## 解題思路
題目要求簡單的條件判斷，根據輸入值輸出不同的結果。程式碼直接使用 `if` 語句進行判斷，並輸出對應的值。雖然題目提示可以使用算術或關係運算子，但使用 `if` 語句是最直觀且易於理解的解決方案。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include<iostream>

using namespace std;

int main(){
	int a;
	cin>>a;
	if(a==0){
		cout<<1;
	}
	else{
		cout << 0;
	}
}
