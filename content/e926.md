# e926 - 跳脫字元

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=e926)

## 題目描述

題目要求輸出一段固定的字串，其中包含一些需要跳脫的字元，例如單引號、反斜線、雙引號等。

## 解題思路

此題為一個簡單的輸出題，只需要按照題目要求，使用適當的跳脫字元，將指定的字串輸出即可。主要需要注意的跳脫字元包括：

*   `'` 需要跳脫成 `\'`
*   `\` 需要跳脫成 `\\`
*   `"` 需要跳脫成 `\"`
*   `|`、`-`、`^`、`(`、`)` 等字元不需要跳脫。

## 複雜度分析

- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <stdio.h>
int main(){
	puts("|'o'|\n\\^_^/\n(\"o\")\n[-%-]");
}
