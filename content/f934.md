# f934 - 消失的二十年

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=f934)

## 題目描述
題目要求計算在給定股票價格序列中，最多買賣一次所能獲得的最大利潤。需要注意的是，必須先買後賣，且不能賣空。

## 解題思路
此題可以使用貪心演算法解決。核心思想是遍歷股票價格序列，記錄遇到的最低價格，並計算當前價格與最低價格之間的差值。在遍歷過程中，不斷更新最大利潤。由於只能買賣一次，因此只需要單次遍歷即可。

## 複雜度分析
- 時間複雜度: O(n)，其中 n 是股票價格序列的長度。因為程式碼只遍歷一次陣列。
- 空間複雜度: O(1)，因為程式碼只使用了常數個額外的變數。

## 程式碼
```cpp
#pragma GCC optimize("Ofast,unroll-loops,no-stack-protector,fast-math")
#pragma GCC target("sse,sse2,sse3,ssse3,sse4,popcnt,abm,mmx,avx,tune=native")
#include <stdio.h>
int n,ma=-1000000,mi=1000000,ans;
inline int read(){
	int a(0);
	char c('0');
	while(c>='0'){
		a=(a<<3)+(a<<1)+c-'0';
		c=getchar_unlocked();
	}
	return a;
}
inline void write(int x) {
	if(x==0)
		putchar_unlocked('0');
	else{
		int stk[9],*ptr(&stk[0]);
		while(x){*ptr=x%10;x/=10;++ptr;}
		while(--ptr>=(&stk[0])){putchar_unlocked(*ptr+'0');}
	}
} 
int main(){
	while(n=read()){
		if(mi>n)mi=n;
		n-=mi;
		if(ans<n)ans=n;
	}
	write(ans);
}
