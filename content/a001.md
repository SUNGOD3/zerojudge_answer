# a001 - 哈囉

## 題目描述
本題要求讀取一行輸入中的一個字串，然後將其與固定的前綴 "hello, " 拼接，並輸出到標準輸出，最後加上一個換行符號。例如，若輸入 "world"，則輸出 "hello, world"。

## 解題思路
這是一個經典的入門級別問題，主要用於練習程式語言的基本輸入與輸出操作。

1.  **引入必要的函式庫**: 使用 C++ 進行標準輸入輸出通常需要 `<iostream>` 函式庫。由於處理字串，也需要 `<string>` 函式庫。
2.  **使用命名空間**: 為了方便使用 `std::cin`, `std::cout`, `std::string` 等，可以使用 `using namespace std;`。
3.  **讀取字串**: 宣告一個 `std::string` 型別的變數來儲存輸入的字串。由於題目描述中提到「輸入總共一行，內含一組文字」，並且範例輸入是單詞，因此使用 `cin >> s` 可以很好地讀取第一個（也是唯一一個）單詞。
4.  **循環讀取處理**: 程式碼使用了 `while(cin >> s)` 這樣的循環結構。這是一種在競爭性程式設計中常見的模式，它會持續從標準輸入讀取字串，直到遇到檔案結尾 (EOF) 或輸入流發生錯誤為止。對於本題，即使輸入只有一行一個單詞，這種寫法也能正確處理，讀取完第一個單詞後，如果沒有更多輸入，循環就會自然終止。
5.  **輸出結果**: 將固定的字串 "hello, " 與讀取到的字串 `s` 組合後輸出。`std::cout << "hello, " << s << std::endl;` 會依序輸出固定前綴、輸入字串，並在結尾加上換行符號。

## 複雜度分析
-   時間複雜度: `O(L)`。其中 `L` 是輸入字串的長度。讀取一個長度為 `L` 的字串需要 `O(L)` 的時間，輸出一個長度為 `L+7` (因為 "hello, " 有 7 個字元) 的字串也需要 `O(L)` 的時間。
-   空間複雜度: `O(L)`。其中 `L` 是輸入字串的長度。主要用於儲存輸入字串 `s`。

## 程式碼
```cpp
#include <iostream>
#include <string> // 包含 string 標頭檔以使用 std::string
using namespace std;
int main() {
    string s;
    while(cin >> s) cout << "hello, " << s << endl;
    return 0;
}

```
