# k391 - 五子棋好難!QAQ

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=k391)

## 題目描述
題目要求根據兩個玩家的棋力值判斷勝負。如果玩家一的棋力值大於玩家二，則玩家一獲勝；如果玩家一的棋力值小於玩家二，則玩家二獲勝；如果棋力值相等，則平局。

## 解題思路
此題為簡單的比較大小問題。程式碼讀取兩個玩家的名字和棋力值，然後使用條件判斷語句（if-else if-else）比較棋力值，並根據比較結果輸出相應的訊息。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
string x,y;
int xv,yv;
int main(){
	cin >> x >> xv >> y >> yv;
	if(xv>yv){
		cout << x << " WIN!";
	}
	else if(xv<yv){
		cout << y << " WIN!";
	}
	else{
		cout << "tie";
	}
}
