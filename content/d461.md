# d461 - 班際籃球賽

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d461)

## 題目描述

題目要求計算在一個單淘汰制的籃球賽中，如果總共有 n 個隊伍參賽，至少需要舉辦多少場比賽才能產生一個冠軍隊伍。題目中給定了兩個規則，但實際上對解題沒有直接影響。

## 解題思路

由於比賽是單淘汰制，要產生一個冠軍隊伍，必須淘汰 n-1 個隊伍。每一場比賽淘汰一個隊伍，因此至少需要 n-1 場比賽。題目中的兩個規則實際上是為了讓題目看起來更複雜，但實際上並不需要考慮。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>

using namespace std;

int main (){
	int n;
	cin >> n;
	cout << n-1 << endl;
	n=0;
}
