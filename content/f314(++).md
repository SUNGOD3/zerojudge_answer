# f314(++) - Error

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=f314(++))

## 題目描述
題目要求計算一個陣列的最大子陣列和，但允許在多個輸入集合中進行計算，並保留所有輸入的最大和。輸入包含多個測試案例，每個案例包含一個陣列。

## 解題思路
這題使用動態規劃的思想。對於每個輸入陣列，我們計算從左到右和從右到左的最大子陣列和。`b[i]` 儲存從左到右的最大子陣列和，`c[i]` 儲存從右到左的最大子陣列和。`ans[i]` 儲存第 i 個位置的最大子陣列和，它是 `b[i]` 和 `c[i]` 中的最大值。在處理完每個輸入陣列後，更新全局最大和 `score`。

## 複雜度分析
- 時間複雜度: O(n*m)，其中 n 是陣列長度，m 是測試案例數量。
- 空間複雜度: O(n)，用於儲存 `a`, `b`, `c`, 和 `ans` 陣列。

## 程式碼
```cpp
#define max( x, y )  ( ((x)>(y)) ? (x):(y) )
#include <stdio.h>
int m,n,a[10001],b[10001],c[10001],ans[10001],score=-1000000,tmp;
inline int read(){
 	int a(0),f(1);
	char c('0');
	while(c>='0'||c=='-'){
		if(c=='-'){
			f=-1;
			c=getchar_unlocked();
		}
		a=(a<<3)+(a<<1)+c-'0';
		c=getchar_unlocked();
	}
	return a*f;
}
int main(){
	m=read();
	n=read();
	while(m--){
		for(int i=0;i<=n;++i){
			a[i]=read();
			b[i]=0;
			c[i]=0;
		}
		b[0]=ans[0]+a[0];
		for(int i=1;i<n;++i)
			b[i]=max(ans[i]+a[i],b[i-1]+a[i]);
		c[n-1]=ans[n-1]+a[n-1];
		for(int i=n-2;i>=0;--i)
			c[i]=max(ans[i]+a[i],c[i+1]+a[i]);
		for(int i=0;i<n;++i)
			ans[i]=max(b[i],c[i]);
	}
	for(int i=0;i<n;++i)
		score=max(score,ans[i]);
	printf("%d",score);
}
