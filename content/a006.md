# a006 - 一元二次方程式

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=a006)

**🏷️ 標籤**: `Mathematics`, `Conditional Logic`

## 題目描述
本題要求解一元二次方程式 `ax^2 + bx + c = 0` 的根。程式需要從標準輸入讀取三個整數 `a, b, c`，然後根據方程式根的類型輸出相應的結果：
1.  如果存在兩個不同的實根，輸出 `Two different roots x1=?? , x2=??`，其中 `x1` 為較大的根，`x2` 為較小的根。
2.  如果存在兩個相同的實根，輸出 `Two same roots x=??`。
3.  如果沒有實根，輸出 `No real root`。
答案可以是浮點數，並且需要遵循指定格式。

## 解題思路
解一元二次方程式 `ax^2 + bx + c = 0` 的核心是使用二次公式：
`x = (-b ± sqrt(b^2 - 4ac)) / (2a)`

這個公式的關鍵部分是判別式 `k = b^2 - 4ac`。判別式的值決定了根的類型：
1.  **當 `k > 0` 時**：方程式有兩個不同的實根。這兩個根分別是 `(-b + sqrt(k)) / (2a)` 和 `(-b - sqrt(k)) / (2a)`。程式需要比較這兩個根的大小，確保較大的根先輸出為 `x1`，較小的根後輸出為 `x2`。
2.  **當 `k = 0` 時**：方程式有兩個相同的實根。此時 `sqrt(k)` 為 `0`，所以兩個根都等於 `(-b) / (2a)`。
3.  **當 `k < 0` 時**：方程式沒有實根，因為無法對負數開平方根得到實數。

程式碼首先讀取 `a, b, c`，然後計算判別式 `k = b*b - 4*a*c`。
接著，使用 `if-else if-else` 結構判斷 `k` 的值：
*   如果 `k > 0`，進入處理兩個不同實根的邏輯。它計算 `x1` 和 `x2` 的值，並通過一個內部 `if` 語句比較它們，確保大者在前輸出。
*   如果 `k == 0`，進入處理兩個相同實根的邏輯，計算並輸出 `x = -b / (2*a)`。
*   如果 `k < 0`，則輸出 `No real root`。

需要注意的是，在進行根的計算時，為了確保浮點數精度，判別式 `k` 和根的計算都使用了 `double` 類型。`sqrt()` 函數也來自 `<math.h>`，返回 `double` 類型。

## 複雜度分析
- 時間複雜度: 每組輸入 `a, b, c`，程式執行固定次數的算術運算（乘法、減法、開方、除法）和條件判斷。因此，解一個一元二次方程式的時間複雜度為 `O(1)`。如果有多組測試資料，則總時間複雜度為 `O(N)`，其中 `N` 為測試資料組數。
- 空間複雜度: 程式只使用了幾個變數 (`a, b, c, k`) 來存儲輸入和中間計算結果。這些變數的數量是常數，與輸入的規模無關。因此，空間複雜度為 `O(1)`。

## 程式碼
```cpp
#include <iostream>
#include <math.h>

using namespace std;

int main (){
	
	int a=0,b=0,c=0;
	
	while(cin >> a >> b >> c){
		double k=(b*b-4*a*c);
		if (k>0){
			cout << "Two different roots x1=";
			if((-b+sqrt(k))/2*a>(-b-sqrt(k))/2*a) {
				cout << (-b+sqrt(k))/2*a << " , x2=" << (-b-sqrt(k))/(2*a) <<endl;
			}
			else{
				cout << (-b-sqrt(k))/2*a << " , x2=" << (-b+sqrt(k))/(2*a) <<endl;
			}
		}
		else if (k==0){
			cout << "Two same roots x=" << -b/(2*a) <<endl;
		}
		else if (k<0){
			cout << "No real root" <<endl;
		}
	} 
	
}
```
