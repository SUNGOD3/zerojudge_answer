# a414 f - Error

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=a414%20f)

## 題目描述
題目要求計算一個非負整數的二進位表示中 1 的個數。輸入為一個非負整數 `a`，輸出 `a` 的二進位表示中 1 的個數。

## 解題思路
這題的解題思路是迭代地檢查輸入數字 `a` 的最低位是否為 1。如果最低位是 1，則將計數器 `ans` 增加 1，然後將 `a` 右移一位。重複此過程直到 `a` 變為 0。右移操作等同於除以 2，因此每次迭代都會檢查 `a` 的下一位。

## 複雜度分析
- 時間複雜度: O(log a)  (因為迴圈執行次數與 `a` 的二進位表示的長度成正比，而二進位表示的長度約為 log2(a))
- 空間複雜度: O(1) (只使用了常數個額外的變數)

## 程式碼
```cpp
#pragma GCC optimize("Ofast,unroll-loops,no-stack-protector,fast-math")
#pragma GCC target("sse,sse2,sse3,ssse3,sse4,popcnt,abm,mmx,avx,tune=native")
#include <stdio.h>
inline int read(){
	int a(0);
	char c('0');
	while(c>='0'){
		a=(a<<3)+(a<<1)+c-'0';
		c=getchar_unlocked();
	}
	return a;
}
inline void write(int x) {
	if(!x){
		putchar_unlocked('0');
		return;
	}
	int stk[3],*ptr(&stk[0]);
	while(x){*ptr=x%10;x/=10;++ptr;}
	while(--ptr>=(&stk[0])){putchar_unlocked(*ptr+'0');}
}
int main(){
	int a,ans;
	while(a=read()){
		ans=0;
		while(a&1){
			++ans;
			a>>=1;
		}
		write(ans);
		putchar_unlocked('\n');
	}
}
