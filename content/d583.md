# d583 - 幼稚的企鵝

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d583)

## 題目描述
題目要求讀取每組測試資料的第一行整數 `n`，代表企鵝的數量。接著讀取第二行包含 `n` 個整數的序列，代表每隻企鵝的座號，這些座號保證是從 1 到 `n` 的不重複數字。最後，程式需要按照座號從小到大輸出排序後的序列。

## 解題思路
這題的解題思路非常簡單，題目要求將輸入的數字序列進行排序後輸出。由於題目中座號的範圍是從 1 到 `n`，且不重複，因此可以直接使用迴圈依序輸出 1 到 `n` 的數字。程式碼直接迴圈輸出 1 到 n，達到排序的效果。

## 複雜度分析
- 時間複雜度: O(n)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std ;
main() {
	int n,i,t;
	while(cin>>n)
	{
		for(i=1; i<=n; i++)
		{cin>>t; cout<<i<<" ";}
		cout<<endl;
	}
	return 0;
}
