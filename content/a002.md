# a002 - 簡易加法

## 題目描述
本題要求讀取兩個以空白分隔的整數 `a` 和 `b`，然後計算並輸出它們的和。輸入的 `a` 和 `b` 絕對值皆小於 `10^6`。

## 解題思路
這是一個基本的輸入輸出練習題。程式只需要使用標準輸入 (cin) 讀取兩個整數 `a` 和 `b`，將它們相加，然後使用標準輸出 (cout) 顯示結果。由於 `a` 和 `b` 的絕對值小於 `10^6`，它們的和最大可能為 `10^6 + 10^6 = 2 * 10^6`，最小可能為 `-10^6 + (-10^6) = -2 * 10^6`。這些值都在標準 `int` 型別的範圍內（通常為 `[-2*10^9, 2*10^9]`）。程式碼中使用 `long long int` 來儲存 `a` 和 `b` 及其和，這是一種更安全的選擇，可以避免潛在的整數溢位問題，即使在本題中 `int` 也足夠。使用 `long long int` 是一種良好的程式設計習慣，尤其是在處理可能較大的數值時。

## 複雜度分析
- 時間複雜度: O(1) - 程式執行固定數量的操作：讀取兩個整數，執行一次加法，然後輸出結果，這些操作不依賴於輸入數值的大小。
- 空間複雜度: O(1) - 程式只需要儲存兩個輸入變數和一個結果，所需的記憶體空間是固定的，不隨輸入數值的大小而變化。

## 程式碼
```cpp
#include <iostream>

using namespace std;

int main (){
	
	long long int a;
	long long int b;
	
	cin >> a >> b;
	
	cout << a+b;
}

```
