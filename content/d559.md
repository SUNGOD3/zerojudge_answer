# d559 - 班號

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d559)

## 題目描述
題目要求讀取多行整數，並針對每一行整數輸出特定格式的字串，字串內容包含固定的前綴字串以及讀取的整數。

## 解題思路
此題為簡單的輸入輸出題。程式需要不斷讀取整數字串，然後按照題目要求的格式輸出字串。由於題目沒有限制輸入的數量，因此使用 `while(cin >> A)` 迴圈來持續讀取輸入。

## 複雜度分析
- 時間複雜度: O(n)，其中 n 是輸入的行數。
- 空間複雜度: O(1)，因為程式只使用了固定大小的變數來儲存輸入字串。

## 程式碼
```cpp
#include <iostream>
using namespace std;

string A;

int main() {
  while (cin >> A){
    cout << "'C' can use printf(\"%d\",n); to show integer like " << A << endl;
  }
}
