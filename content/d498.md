# d498 - 我不說髒話

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d498)

## 題目描述
題目要求輸出指定次數的字串 "I don't say swear words!"。輸入為一個正整數 n，表示需要輸出的次數。

## 解題思路
此題為一個簡單的迴圈應用。程式接收一個整數 n 作為輸入，然後使用一個 while 迴圈重複輸出 "I don't say swear words!" 總共 n 次。迴圈條件為 n > 0，每次迴圈結束後 n 減 1，直到 n 變為 0 迴圈結束。

## 複雜度分析
- 時間複雜度: O(n)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>

using namespace std;

int main (){
	int n;
	cin >> n;
	while(n>0){
	cout << "I don't say swear words!" << endl;
	n--;}
	n=0;
}
