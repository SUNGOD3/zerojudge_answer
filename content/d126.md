# d126 - 一、牧场围栏

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d126)

## 題目描述
題目要求計算一個矩形牧場的周長，給定矩形的長 `a` 和寬 `b`。

## 解題思路
矩形的周長公式為 `2 * (長 + 寬)`。程式只需要讀取長和寬，然後根據公式計算周長並輸出即可。題目有多組測試資料，需要使用 `while` 迴圈讀取每一組資料。

## 複雜度分析
- 時間複雜度: O(n)，其中 n 是測試資料組數。
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>

using namespace std;

int main(){
	long long int a,b;
	while(cin >> a >> b){
		cout << a*2+b*2 << endl;
	}
}
