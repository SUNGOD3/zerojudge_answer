# f429 - 高雄市109年資訊競賽國中組第七題

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=f429)

## 題目描述
題目要求讀取三個字串 `a`、`b` 和 `c`，然後按照 `b`、`c`、`a` 的順序輸出它們，每個字串佔一行。

## 解題思路
此題為簡單的輸入輸出題，只需要按照題目要求讀取三個字串，並按照指定順序輸出即可。沒有複雜的演算法或資料結構需求。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
string a,b,c;
int main(){
	cin >> a >> b >> c;
	cout << b << "\n" << c << "\n" << a;
}
