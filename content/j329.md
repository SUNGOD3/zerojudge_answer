# j329 - 絕交

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=j329)

## 題目描述
題目要求根據輸入的 A 的名字、B 的名字以及 A 是否想和 B 絕交的訊息，輸出相應的句子。如果 A 想和 B 絕交，則輸出 "A 想和 B 絕交"；否則，輸出 "A 不想和 B 絕交"。

## 解題思路
這題是一個簡單的條件判斷問題。程式需要讀取三個字串：A 的名字、B 的名字以及一個表示 A 是否想和 B 絕交的字串（"yes" 或 "no"）。然後，根據這個字串的值，使用 `if` 語句來判斷是否輸出 "A 想和 B 絕交" 或 "A 不想和 B 絕交"。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <bits/stdc++.h>
using namespace std;
string x,y,z;
int main(){
	cin >> x >> y >> z;
	if(z=="yes"){
		cout << x << "想和" << y << "絕交"; 
	}
	else{
		cout << x << "不想和" << y << "絕交"; 
	}
}
