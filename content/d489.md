# d489 - 伏林的三角地

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d489)

## 題目描述
題目要求計算給定三邊長三角形的面積，並將面積作為土地價值輸出。

## 解題思路
本題為數學題，利用海龍公式計算三角形面積。海龍公式如下：
面積 = sqrt(s * (s - a) * (s - b) * (s - c))，其中 s = (a + b + c) / 2，a, b, c 為三角形的三邊長。
程式碼直接套用海龍公式計算面積，並輸出結果。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>

using namespace std; 

int main (){
	
	int a,b,c=0;
	int s=0;
	int ans=0;
	while(cin >> a >> b >> c){
		s=0;
		s=(a+b+c)/2;
		ans=s*(s-a)*(s-b)*(s-c);
		cout << ans;
	}	
}
