# k632 - 產生隨機亂數2

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=k632)

## 題目描述

題目要求輸出一個由 1 到 N 的整數序列，以空格分隔。序列的內容為 1, 2, 3, ..., N。

## 解題思路

題目本身並不需要產生隨機數列，而是要求按照順序輸出 1 到 N 的整數，並用空格分隔。程式碼直接迴圈輸出即可。程式碼使用了優化的輸入輸出方式，`getchar_unlocked()` 和 `putchar_unlocked()` 避免了標準 I/O 的同步問題，提升了效率。`read()` 和 `write()` 函數分別用於快速讀取整數和輸出整數。

## 複雜度分析
- 時間複雜度: O(n)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <stdio.h>
int n;
inline int read(){
	int a(0);
	char c('0');
	while(c>='0'){
		a=(a<<3)+(a<<1)+c-'0';
		c=getchar_unlocked();
	}
	return a;
}
inline void write(int x) {
	if(x==0)
		putchar_unlocked('0');
	else{
		int stk[9],*ptr(&stk[0]);
		while(x){*ptr=x%10;x/=10;++ptr;}
		while(--ptr>=(&stk[0])){putchar_unlocked(*ptr+'0');}
	}
}
int main(){
	n=read();
	for(int i=1;i<=n;++i){
		write(i);
		putchar_unlocked(' ');
	}
}
