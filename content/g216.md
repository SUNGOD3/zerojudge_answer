# g216 - 2^(10^n)

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=g216)

## 題目描述
題目要求根據輸入的整數 `n`，輸出字串 "2^(10^" + `n` + ")"。

## 解題思路
題目非常簡單，只需要讀取輸入的整數 `n`，然後按照題目要求的格式輸出字串即可。不需要進行任何計算。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int main(){
	int n;
	while(cin >> n){
		cout << "2^(10^" << n << ")";
	}
}
