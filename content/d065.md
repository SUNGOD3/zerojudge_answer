# d065 - 三人行必有我師 (1 行版)

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d065)

## 題目描述
題目要求讀取三個非負整數，並輸出其中最大的那個。

## 解題思路
此題為簡單的比較問題。程式碼透過一系列的 if-else 判斷式，比較三個輸入數字的大小，並輸出最大的數字。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>

using namespace std;

int main (){
	int a;
	int b;
	int c;
	while(cin >> a >> b >> c){
		if(a>=b&&a>=c){
			cout << a << endl;
		}
		else if(b>=a&&b>=c){
			cout << b << endl;
		}
		else{
			cout << c << endl;
		}
	}
	a=0;
	b=0;
	c=0;
}
