# k689 - 肥余歷險記---(反數學城3)

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=k689)

## 題目描述
題目要求計算肥余睡覺後血量的增加值，並將其加到初始血量上，最後輸出總血量。輸入包含測資個數、初始血量、每小時增加的血量以及睡眠時間。

## 解題思路
題目是一個簡單的算術問題。對於每個測資，我們需要讀取初始血量 `a`、每小時增加的血量 `b` 和睡眠時間 `c`。然後，計算睡眠期間增加的血量 `b * c`，並將其加到初始血量 `a` 上，得到最終血量 `a + b * c`。最後，輸出最終血量。

## 複雜度分析
- 時間複雜度: O(n)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
long long n,a,b,c;
int main(){
	cin >> n;
	for(int i=0;i<n;++i){
		cin >> a >> b >> c;
		cout << a+b*c << "\n";
	}
}
