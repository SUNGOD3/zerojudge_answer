# c657 - 最長連續字母

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=c657)

## 題目描述

題目要求找出給定字串中最長連續出現的字母，並輸出該字母及其出現次數。若有多個連續字母出現次數相同，則輸出最先出現的字母。

## 解題思路

此題可以使用貪婪演算法解決。遍歷字串，記錄目前連續字母的數量。如果當前字母與前一個字母相同，則連續數量加一。如果不同，則將連續數量重置為一。在遍歷過程中，記錄目前最長連續字母及其數量。如果發現更長的連續字母，則更新最長連續字母及其數量。

## 複雜度分析
- 時間複雜度: O(n)，其中 n 是字串的長度。需要遍歷字串一次。
- 空間複雜度: O(1)，只需要常數額外的空間來儲存目前連續字母的數量、最長連續字母及其數量。

## 程式碼
```cpp
#include <iostream>
#include <string>
using namespace std;
int main(){
	string a;
	while(cin>>a){
		int max=0,n=1;
		char ans;
		for(int i=0;i<a.length()-1;i++){
			if(a[i]==a[i+1]){
				n++;
				if(n>max){
					max=n;
					ans=a[i];
				}
			}
			else
				n=1;
		}
		cout << ans << " " << max << endl;
	}
}
