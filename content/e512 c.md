# e512 c - Error

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=e512%20c)

## 題目描述
題目要求計算兩個矩形的交集矩形的左上角和右下角座標。給定兩個矩形的左上角和右下角座標，需要判斷它們是否相交，並計算交集矩形的座標。如果兩個矩形不相交，則輸出基於特定條件計算的座標。

## 解題思路
題目描述了四種情況：
1. 兩個矩形完全重疊（左上角相同）。
2. 兩個矩形右下角相同。
3. 兩個矩形左上角相同。
4. 兩個矩形不重疊，需要計算一個新的矩形座標。

程式碼根據這四種情況進行判斷，並輸出相應的座標。核心邏輯是比較兩個矩形的座標，以確定它們的相對位置，然後根據不同的位置計算交集矩形的座標。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <stdio.h>
int main(){
	double a,b,c,d,e,f,g,h;
	while(scanf("%lf%lf%lf%lf%lf%lf%lf%lf",&a,&b,&c,&d,&e,&f,&g,&h)>0){
		if(a==e&&b==f)printf("%.3lf %.3lf\n",c+g-a,d+h-b);
		else if(c==g&&d==h)printf("%.3lf %.3lf\n",a+e-c,b+f-d);
		else if(a==g&&b==h)printf("%.3lf %.3lf\n",c+e-a,d+f-b);
		else printf("%.3lf %.3lf\n",a+g-c,b+h-d);
	}
}
