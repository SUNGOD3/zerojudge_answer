# g496 - 彗星列車 (Comet)

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=g496)

## 題目描述
題目描述：有一條長度為 `y` 的彗星列車，每節車廂長度為 `x`。求至少需要多少節車廂才能覆蓋整條彗星列車。

## 解題思路
題目要求計算 `y` 除以 `x` 的結果，如果餘數不為 0，則需要額外一節車廂。因此，可以使用整數除法和取餘運算來解決這個問題。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
long long x,y;
int main(){
	cin.tie(0); ios::sync_with_stdio(false);
	cin >> x >> y;
	cout << y/x+(y%x!=0);
}
