# d049 - 中華民國萬歲！

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d049)

## 題目描述
題目要求將輸入的西元年份轉換為民國年份，計算方式為西元年份減去 1911。輸入的西元年份範圍是 1912 到 2147483647。

## 解題思路
這題是一個簡單的數學計算問題。由於民國年份的起算點是 1912 年，因此只需要將輸入的西元年份減去 1911 即可得到民國年份。題目給定的輸入範圍保證了計算結果不會超出 long long int 的範圍。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>

using namespace std;

int main() {
long long int a;
cin>>a;
cout<<a-1911;
}
