# d202 - a+b??

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d202)

## 題目描述
題目要求計算兩個浮點數 a 和 b 的和，並將結果四捨五入到小數點後 200 位。由於輸入的數字可能非常大，標準的浮點數類型無法精確表示，因此需要使用字串來處理數字。

## 解題思路
程式碼使用字串來表示浮點數，並模擬手動加法的過程。它將字串分割為整數部分和小數部分，分別進行加法運算。整數部分和分數部分都使用陣列來儲存每一位數字，然後從最低位開始進行加法，處理進位。分數部分加法完成後，將整數部分和小數部分合併，並處理最終的進位。最後，將結果輸出到小數點後 200 位。程式碼使用了 `putchar_unlocked` 來提高輸出效率。

## 複雜度分析
- 時間複雜度: O(N)，其中 N 是輸入字串的長度。因為程式碼需要遍歷字串的每一個字符來進行加法運算。
- 空間複雜度: O(N)，因為程式碼使用陣列來儲存數字的每一位，陣列的大小與輸入字串的長度成正比。

## 程式碼
```cpp
#pragma GCC optimize("Ofast,unroll-loops,no-stack-protector,fast-math")
#pragma GCC target("sse,sse2,sse3,ssse3,sse4,popcnt,abm,mmx,avx,tune=native")
#include <iostream>
using namespace std;
inline void write(int x) {
	if(x==0){
		putchar_unlocked('0');
		return;
	}
	int stk[9],*ptr(&stk[0]);
	while(x){*ptr=x%10;x/=10;++ptr;}
	while(--ptr>=(&stk[0])){putchar_unlocked(*ptr+'0');}
}
inline void fadd(string x,string y){
	int xl=x.length(),yl=y.length(),xf=0,yf=0,w=200,fsum[251]={0},sum[251]={0};
	while(x[xf]!='.')++xf;
	while(y[yf]!='.')++yf;
	for(int i=xf+1,it=0;i<xl;++i,++it)fsum[it]=x[i]-48;
	for(int i=yf+1,it=0;i<yl;++i,++it)fsum[it]+=y[i]-48;
	if(fsum[200]>4){
		fsum[200]=0;
		++fsum[199];
	}
	for(int i=200;i;--i){
		if(fsum[i]>9){
			fsum[i]-=10;
			++fsum[i-1];
		}
	}
	if(fsum[0]>9)sum[0]=1;
	for(int i=xf-1,it=0;i>=0;--i,++it)sum[it]=x[i]-48;
	for(int i=yf-1,it=0;i>=0;--i,++it)sum[it]+=y[i]-48;
	while(sum[w]==0&&w)--w;
	for(int i=0;i<=w;++i){
		if(sum[i]>9){
			sum[i]-=10;
			++sum[i+1];
		}
	}
	if(sum[w+1])++w;
	for(int i=w;i>=0;--i)write(sum[i]);
	putchar_unlocked('.');
	for(int i=0;i<200;++i)write(fsum[i]);
}
int main(){
	cin.tie(0); ios::sync_with_stdio(false);
	char t;
	while(1){
		string a,b;
		t=getchar_unlocked();
		while((t>='0'&&t<='9')||t=='.'){
			a+=t;
			t=getchar_unlocked();
		}
		t=getchar_unlocked();
		while((t>='0'&&t<='9')||t=='.'){
			b+=t;
			t=getchar_unlocked();
		}
		if(t==-1)break;
		fadd(a,b);
		putchar_unlocked('\n');
	}
}
