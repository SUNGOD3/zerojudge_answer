# b294 - 經濟大恐荒

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=b294)

## 題目描述

題目要求計算文文在 n 天內購買饅頭所花費的總金額。每天饅頭的價格隨著天數增加而線性增長，文文每天購買的饅頭數量給定，需要計算總花費。

## 解題思路

這道題可以使用模擬的方法解決。題目給定了每天購買的饅頭數量以及每天的價格，因此可以遍歷每一天，計算每天的花費（饅頭數量 * 價格），然後將每天的花費累加到總花費中。由於價格每天遞增，因此可以視為一個貪心策略，每天都以當天的價格購買。

## 複雜度分析
- 時間複雜度: O(n)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int main(){
	int n,ans=0,x;
	cin >> n;
	for(int i=1;i<=n;++i){
		cin >> x;
		ans+=i*x;
	}
	cout << ans ;
}
