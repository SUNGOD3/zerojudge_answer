# f708 - 蟲蟲危機 (Insect)

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=f708)

## 題目描述
題目描述一開始有 n 隻蟲蟲，接著有 m 隻蟲蟲被吃掉。要求判斷剩下的蟲蟲數量是否大於 0 且大於被吃掉的蟲蟲數量。如果滿足條件，輸出 "Yes"，否則輸出 "No"。

## 解題思路
這題的解題思路非常直接。首先讀取蟲蟲的初始數量 n 和被吃掉的蟲蟲數量 m。然後，讀取 n 個數字，計算蟲蟲的總和，再讀取 m 個數字，從總和中減去這些數字。最後，判斷剩下的蟲蟲數量是否大於 0 且大於 m。

## 複雜度分析
- 時間複雜度: O(n + m)
- 空間複雜度: O(1)

## 程式碼
```cpp
#pragma GCC optimize("Ofast,unroll-loops,no-stack-protector,fast-math")
#pragma GCC target("sse,sse2,sse3,ssse3,sse4,popcnt,abm,mmx,avx,tune=native")
#include <stdio.h>
inline int read(){
	int a(0);
	char c('0');
	while(c>='0'){
		a=(a<<3)+(a<<1)+c-'0';
		c=getchar_unlocked();
	}
	return a;
}
inline void write(int x) {
	if(x==0)
		putchar_unlocked('0');
	else{
		int stk[9],*ptr(&stk[0]);
		while(x){*ptr=x%10;x/=10;++ptr;}
		while(--ptr>=(&stk[0])){putchar_unlocked(*ptr+'0');}
	}
}
int main(){
	int n,m,sum=0,tmp;
	n=read();
	m=read();
	for(int i=0;i<n;++i){
		tmp=read();
		sum+=tmp;
	} 
	for(int j=0;j<m;++j){
		tmp=read();
		sum-=tmp;
	}
	if(sum<=0||n<=m){
		putchar_unlocked('N');
		putchar_unlocked('o');
	}
	else{
		putchar_unlocked('Y');
		putchar_unlocked('e');
		putchar_unlocked('s');
	}
}
