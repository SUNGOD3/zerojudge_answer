# e357++ - 遞迴函數練習

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=e357++)

## 題目描述
題目定義了一個遞迴函數 F(x)，其值根據 x 的奇偶性以及 x 是否為 1 進行計算。目標是給定一個正整數 x，計算 F(x) 的值。

## 解題思路
這題的核心在於理解遞迴的定義並有效地實現它。直接使用遞迴可能會導致重複計算，因此使用備忘錄 (memoization) 技術來優化效能。備忘錄是一種動態規劃的技巧，它將已經計算過的結果儲存起來，避免重複計算。程式碼中 `arr` 陣列用來儲存計算過的 F(x) 值。當函數被呼叫時，首先檢查 `arr[x]` 是否已經有值，如果有，則直接返回儲存的值；否則，計算 F(x) 的值，並將其儲存到 `arr[x]` 中，然後返回。

## 複雜度分析
- 時間複雜度: O(x)
- 空間複雜度: O(x)

## 程式碼
```cpp
#include <iostream>
//#include <stdlib.h>
using namespace std;
int arr[2000000] = {0};
int fun(int x){
	if(arr[x])
		return arr[x];
	if(x==1){
		arr[x]=1;
		return 1; 
	}
	if(x%2==0){
		arr[x] = fun(x/2); 
		return arr[x];
	} 
	arr[x-1] = fun(x-1);
	arr[x+1] = fun(x+1);
	return arr[x-1]+arr[x+1];
}

int main(){
	int x=0;
//	memset(arr,10000,-1);
	while(cin >> x){
		cout << fun(x);
	}
	return 0;
}
