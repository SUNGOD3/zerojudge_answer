# i722 - 蝸牛好蟀

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=i722)

## 題目描述
題目要求輸出固定的字串 "Snail is \"handsome.\""。

## 解題思路
此題目為一個簡單的輸出題，不需要任何輸入處理或計算邏輯。程式碼直接輸出題目要求的字串即可。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int main(){
	cout << "Snail is \"handsome.\"";
}
