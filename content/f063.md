# f063 - The Strongest Chain

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=f063)

## 題目描述
題目給定若干條鍊條，每條鍊條包含多個環節，每個環節都有一個強度值。目標是找出所有鍊條中，最強的鍊條強度，其中鍊條的強度定義為其所有環節中強度最小的環節強度。

## 解題思路
對於每一條鍊條，我們需要找到其最小環節強度。然後，在所有鍊條的最小強度中，找到最大的那個值，即為最強鍊條的強度。程式碼使用迴圈迭代每一條鍊條，並在內部迴圈中找到該鍊條的最小強度。最後，使用 `std::max` 函數更新最強鍊條強度。

## 複雜度分析
- 時間複雜度: O(n*k)，其中 n 是鍊條的數量，k 是每條鍊條的環節數量。因為需要遍歷每一條鍊條，並且在每一條鍊條中找到最小的環節強度。
- 空間複雜度: O(1)，程式碼只使用了常數級別的額外空間。

## 程式碼
```cpp
#pragma GCC optimize("Ofast,unroll-loops,no-stack-protector,fast-math")
#pragma GCC target("sse,sse2,sse3,ssse3,sse4,popcnt,abm,mmx,avx,tune=native")
#include <stdio.h>
#include <algorithm>
inline int read(){
	int a(0);
	char c('0');
	while(c>='0'){
		a=(a<<3)+(a<<1)+c-'0';
		c=getchar_unlocked();
	}
	return a;
}
inline void write(int x) {
	if(x==0)
		putchar_unlocked('0');
	else{
		int stk[9],*ptr(&stk[0]);
		while(x){*ptr=x%10;x/=10;++ptr;}
		while(--ptr>=(&stk[0])){putchar_unlocked(*ptr+'0');}
	}
}
int main(){
	int n=read(),m,ans=0,k;
	while(n--){
		m=read();
		int tmp=999999999;
		while(m--){
			k=read();
			tmp=std::min(k,tmp);
		}
		ans=std::max(ans,tmp);
	}
	write(ans);
}
