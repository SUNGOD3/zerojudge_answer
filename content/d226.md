# d226 - 10071 - Back to High School Physics

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d226)

## 題目描述
題目要求計算一個以初速度 v 和等加速度運動的粒子在 2t 秒後所經過的位移。已知 t 秒後的末速度為 v。

## 解題思路
根據物理公式，末速度 v = v0 + at，其中 v0 是初速度，a 是加速度，t 是時間。
題目給定 v 和 t，我們可以求出加速度 a = (v - v0) / t。
由於粒子從靜止開始運動，初速度 v0 = 0，因此 a = v / t。
然後，我們可以使用位移公式 s = v0*t + 0.5*a*t^2 計算 2t 秒後的位移。
由於 v0 = 0，位移公式簡化為 s = 0.5 * a * (2t)^2 = 0.5 * (v/t) * 4t^2 = 2 * v * t。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>

using namespace std;

int main(){

	long long int v=0,t=0;
	while(cin >> v >> t){
		cout << v*2*t << endl;
	}

}
