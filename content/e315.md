# e315 - NOIP2017 1.成绩

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=e315)

## 題目描述
題目要求計算牛牛的 C++ 課程總成績，總成績由作業成績（20%）、小測成績（30%）和期末考試成績（50%）加權計算得出。

## 解題思路
題目描述了總成績的計算公式，直接根據公式計算即可。輸入三個浮點數，分別代表作業成績、小測成績和期末考試成績，然後按照給定的權重計算總成績，並輸出結果。由於題目要求輸出整數，需要將計算結果四捨五入到最接近的整數。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <stdio.h>
int main(){
	float a,b;
	while(scanf("%f%f",&a,&b)>0){
		a*=0.2;
		a+=b*0.3;
		scanf("%f",&b);
		printf("%.0f\n",a+b*0.5);
	}
}
