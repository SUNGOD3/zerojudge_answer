# s142 - 最大正方形

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=s142)

## 題目描述
題目要求在一個 n x m 的二元矩陣中，找出邊長最大的全為 1 的正方形，並輸出其邊長。矩陣中保證至少有一個 1。

## 解題思路
本題使用動態規劃和前綴和的結合來解決。首先，計算一個前綴和矩陣 `dp[i][j]`，其中 `dp[i][j]` 儲存了矩陣中 `(1, 1)` 到 `(i, j)` 的所有元素的和。然後，對於每個可能的正方形邊長 `k`，檢查以 `(i, j)` 為右下角的正方形是否全為 1。這可以通過使用前綴和快速計算正方形內的元素和來完成。如果正方形內的元素和等於 `k*k`，則說明該正方形全為 1，更新最大正方形的邊長。

## 複雜度分析
- 時間複雜度: O(n*m*min(n,m))
- 空間複雜度: O(n*m)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int dp[1001][1001], n, m, ans;
int main(){
	cin.tie(0); ios::sync_with_stdio(false);
	cin >> n >> m;
	for(int i=1;i<=n;++i){
		for(int j=1;j<=m;++j){
			cin >> dp[i][j];
			dp[i][j]+=dp[i-1][j]+dp[i][j-1]-dp[i-1][j-1];
			for(int k=ans+1;k<=min(i, j);++k){
				if(dp[i][j]-dp[i-k][j]-dp[i][j-k]+dp[i-k][j-k]==k*k){
					ans=k;
				}		
				else{
					break;
				}	
			}
		}
	}
	cout << ans;
}
