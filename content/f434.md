# f434 - 458 - The Decoder

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=f434)

## 題目描述
題目要求將加密的字串解碼成原始字串。加密方式是將每個字元的 ASCII 值減去 7。

## 解題思路
題目描述簡單明瞭，直接讀取輸入字串，對每個字元執行減 7 的操作，然後輸出解碼後的字串。需要注意的是，題目要求處理換行符號，換行符號不進行解碼，直接輸出。使用 `getchar_unlocked()` 和 `putchar_unlocked()` 可以提高 I/O 效率。

## 複雜度分析
- 時間複雜度: O(n)，其中 n 是輸入字串的長度。
- 空間複雜度: O(1)，只需要常數級別的額外空間。

## 程式碼
```cpp
#pragma GCC optimize("Ofast,unroll-loops,no-stack-protector,fast-math")
#pragma GCC target("sse,sse2,sse3,ssse3,sse4,popcnt,abm,mmx,avx,tune=native")
#include <stdio.h>
int main(){
	char a;
	while(a=getchar_unlocked()){
		if(a==-1)break;
		if(a=='\n'){
			putchar_unlocked('\n');
			continue;
		}
		putchar_unlocked(a-7);
	}
}
