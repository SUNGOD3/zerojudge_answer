# d463 - 挑战极限 Part 5：开立方根

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d463)

## 題目描述
題目要求輸入一個正整數 N (0 < N < 2^64)，並輸出 N 的立方根。

## 解題思路
此題的解題思路非常直接，利用 `pow` 函數計算輸入數字的 1/3 次方，即可得到立方根。由於題目要求輸出整數部分，因此使用 `printf("%.0lf\n", ...)` 格式化輸出，保留小數點後 0 位。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include<stdio.h>
#include<stdlib.h>
#include<math.h>
main()
{
 unsigned long long int n;
 while(scanf("%llu",&n)==1)
     printf("%.0lf\n",pow(n,1/3.0));
  return 0;
}
