# d195(d&c) - Counting Inversions

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=d195(d&c))

## 題目描述
題目要求計算給定數列的 inversion count，也就是數列中滿足 `i < j` 且 `a[i] > a[j]` 的數對 (i, j) 的數量。如果 inversion count 是奇數，輸出 "Marcelo"，否則輸出 "Carlos"。

## 解題思路
這題使用分治法 (Divide and Conquer) 來計算 inversion count。核心思想是將數列分成左右兩部分，遞迴計算左半部分和右半部分的 inversion count，然後計算跨越左右兩部分的 inversion count。跨越部分的 inversion count 可以通過對右半部分排序後，對於左半部分的每個元素，使用二分搜尋找到右半部分中小於該元素的數量來計算。

具體步驟如下：

1.  將數列分成左右兩部分。
2.  遞迴計算左半部分的 inversion count。
3.  遞迴計算右半部分的 inversion count。
4.  對右半部分排序。
5.  對於左半部分的每個元素，使用二分搜尋在排序後的右半部分中找到小於該元素的數量，將其加到總的 inversion count 中。
6.  返回左半部分 inversion count + 右半部分 inversion count + 跨越部分的 inversion count。

## 複雜度分析
- 時間複雜度: O(n log^2 n)。分治的過程是 O(log n) 層，每一層需要排序 O(n log n) 的時間，以及二分搜尋 O(n log n) 的時間，因此總時間複雜度是 O(n log^2 n)。
- 空間複雜度: O(n)。主要來自遞迴調用的堆疊空間和排序時使用的額外空間。

## 程式碼
```cpp
#include <iostream>
#include <algorithm>
#include <memory.h>
using namespace std;
int a[100005],n;
inline long long int dam(int l,int r){
	int mid=(l+r)/2;
	if(l+1>=r)return 0;
	long long int ans=dam(l,mid)+dam(mid,r);
	sort(a+mid,a+r);
	for(int i=l;i<mid;++i)
		ans+=lower_bound(a+mid,a+r,a[i])-(a+mid);
	return ans;
}
int main(){
	cin.tie(0); ios::sync_with_stdio(false);
	while(cin >> n){
		if(n==0)break;
		memset(a,0,sizeof(a));
		for(int i=0;i<n;++i)
			cin >> a[i];
		int ac=dam(0,n);
		if(ac%2)
			cout << "Marcelo\n";
		else
			cout << "Carlos\n";
	}
}
