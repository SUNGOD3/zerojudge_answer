# g489 - 社團點點名

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=g489)

## 題目描述
題目要求計算社團缺席人數。給定社團總人數 `n` 和出席人數 `m`，直接計算 `n - m` 即可得到缺席人數。題目中關於輸入學號的部分是誤導資訊，實際上並不需要處理學號。

## 解題思路
題目描述中給定的輸入包含總人數、出席人數，以及出席學號和總學號。然而，題目只需要計算缺席人數，因此只需要讀取總人數和出席人數，然後進行簡單的減法運算即可。程式碼直接讀取 `n` 和 `m`，並輸出 `n - m`。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int n,m;
int main(){
	cin.tie(0); ios::sync_with_stdio(false);
	cin >> n >> m;
	cout << n-m;
}
