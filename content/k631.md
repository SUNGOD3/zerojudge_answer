# k631 - 解一元一次方程

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=k631)

## 題目描述

題目要求解一個簡單的一元一次方程，形式為 `x = z`，其中 `x` 是未知數，`z` 是乘積。程式需要讀取一個數字 `x`，一個未知數 `y`，一個等號 `=`，以及一個乘積 `z`，然後計算出 `y = z / x` 的結果並輸出。

## 解題思路

這道題非常簡單，只需要讀取輸入，進行一次除法運算，然後輸出結果即可。題目保證輸入的方程式只包含乘法，因此不需要考慮其他運算符。由於題目中未知數是字元，所以直接輸出即可。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int x,z;
char y,c;
int main(){
	cin >> x >> y >> c >> z;
	cout << y << " = " << z/x; 
}
