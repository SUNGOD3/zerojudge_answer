# o254 - 我要變得更強!

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=o254)

## 題目描述
題目要求計算學妹至少需要將強度提升至多少才能比臨末強。如果學妹已經比臨末強，則輸出學妹目前的強度。否則，輸出臨末的強度加一。

## 解題思路
題目非常簡單，只需要比較學妹和臨末的強度，並根據比較結果輸出不同的值。如果學妹的強度大於或等於臨末的強度，則輸出學妹的強度。否則，輸出臨末的強度加一。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int main(){
	int x,y;
	cin >> x >> y;
	if(y>=x)cout << y+1;
	else cout << x;
}
