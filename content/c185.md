# c185 - Hey Jude

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=c185)

## 題目描述
題目要求讀取一個名字，並將其替換到 "Hey Jude" 中的 "Jude" 位置，輸出 "Hey [輸入的名字]"。

## 解題思路
這題是一個簡單的字串處理問題。只需要讀取一行輸入，然後將輸入的字串連接到 "Hey " 前面，最後輸出即可。

## 複雜度分析
- 時間複雜度: O(n)，其中 n 是輸入字串的長度。
- 空間複雜度: O(n)，用於儲存輸入字串。

## 程式碼
```cpp
#include <iostream>
using namespace std;
string s;
int main(){
	getline(cin,s);
	cout << "Hey " << s;
}
