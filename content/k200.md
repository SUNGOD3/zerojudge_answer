# k200 - 三人行必有我師

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=k200)

## 題目描述
題目要求讀取三個非負整數，並輸出其中最大的那個。

## 解題思路
此題為簡單的比較問題。直接使用 `max` 函數即可找到三個數字中的最大值。程式碼中先讀取三個整數 `x`、`y` 和 `z`，然後使用 `max(x, max(y, z))` 找出最大值並輸出。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int x,y,z;
int main(){
	cin >> x >> y >> z;
	cout << max(x,max(y,z));
}
