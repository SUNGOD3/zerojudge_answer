# a148 - You Cannot Pass?!

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=a148)

## 題目描述
題目要求讀取 n 科考試成績，計算平均分數，如果平均分數大於 59，則輸出 "no"，否則輸出 "yes"。

## 解題思路
這題是一個簡單的模擬題。程式首先讀取科目數 n，然後讀取 n 個成績，計算總分，再計算平均分。最後根據平均分是否大於 59 輸出結果。

## 複雜度分析
- 時間複雜度: O(n)
- 空間複雜度: O(n)

## 程式碼
```cpp
#include <iostream>

using namespace std;

int main (){
	
	double g=0;
	double s=0;
	int n=0;
	
	
	while(cin >> n){
		int a[n];
		for(int i=0;i<n;i++){
			cin >> a[i];
			g+=a[i];
		}
		s=g/n;
		if(s>59){
			cout << "no" << endl;
		}
		else{
			cout << "yes" << endl;
		}
		n=0;
		g=0;
		s=0;
	}
}
