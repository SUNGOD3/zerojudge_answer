# c418 - Bert的三角形 (1)

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=c418)

## 題目描述
題目要求輸出一個 n 層的三角形，其中第 i 層包含 i 個星號 (*)。

## 解題思路
此題為一個簡單的輸出格式問題。使用巢狀迴圈即可解決。外層迴圈控制層數，內層迴圈控制每層輸出的星號數量。

## 複雜度分析
- 時間複雜度: O(n^2)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>

using namespace std;

int main (){
	
	int n=0;
	while(cin >> n){
		
		for(int i=1;i<=n;i++){
			for(int j=1;j<=i;j++){
				cout << '*' ;
			}
			cout << endl;
		}
		
	}
	
	
}
