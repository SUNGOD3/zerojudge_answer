# f775 - 我只是個輸出。

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=f775)

## 題目描述
題目要求輸出字串 "_xdddd"。

## 解題思路
題目非常簡單，只需要使用 `cout` 輸出指定的字串即可。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int main(){
	cout << "_xdddd";
}
