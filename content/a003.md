# a003 - 兩光法師占卜術

## 題目描述
這道題目要求我們根據一個特定的占卜規則，輸入月份 `M` 和日期 `D`，然後計算出對應的運勢。占卜規則定義了一個公式：`S = (M * 2 + D) % 3`。根據 `S` 的值，判斷並輸出三種不同的運勢：
- 當 `S = 0` 時，運勢為「普通」。
- 當 `S = 1` 時，運勢為「吉」。
- 當 `S = 2` 時，運勢為「大吉」。
程式需要不斷讀取輸入的 `M` 和 `D`，直到輸入結束。

## 解題思路
這是一道非常基礎的數學計算與條件判斷題。解題的核心步驟如下：
1. **讀取輸入**：使用 `while(cin >> M >> D)` 迴圈，持續讀取月份 `M` 和日期 `D`。這種方式會一直讀取直到遇到檔案結束符號 (EOF) 或無效輸入。
2. **計算 S 值**：根據題目給定的公式 `S = (M * 2 + D) % 3` 來計算出 `S` 的值。這裡涉及基本的乘法、加法和取模 (modulo) 運算。
3. **條件判斷與輸出**：根據計算出的 `S` 值，使用 `if` 語句進行判斷：
    - 如果 `S` 等於 0，則輸出字串 "普通"。
    - 如果 `S` 等於 1，則輸出字串 "吉"。
    - 如果 `S` 等於 2，則輸出字串 "大吉"。
由於 `S` 的值只可能為 0、1 或 2，所以三個 `if` 語句可以獨立判斷，也可以組織成 `if-else if-else` 結構。程式碼中使用的是三個獨立的 `if` 判斷，對於這種互斥的情況，其效果與 `if-else if-else` 相同。

## 複雜度分析
- 時間複雜度: `O(1)` (針對每組輸入)。對於每對輸入的 `M` 和 `D`，程式執行固定的算術運算和條件判斷，其時間消耗是常數級別。如果總共有 `N` 組輸入，則總時間複雜度為 `O(N)`。
- 空間複雜度: `O(1)`。程式只使用了兩個整數變數 `M` 和 `D` 來儲存當前的輸入值，不論輸入數據量多大，所需的記憶體空間都是固定的，因此空間複雜度為常數級別。

## 程式碼
```cpp
#include <iostream>

using namespace std;

int main (){
	
	int M;
	int D;
	
	while(cin >> M >> D){
		if((M*2+D)%3==0){
			cout << "普通" << endl;
		}
		if((M*2+D)%3==1){
			cout << "吉" << endl;
		}
		if((M*2+D)%3==2){
			cout << "大吉" << endl;
		}
	}
}

```
