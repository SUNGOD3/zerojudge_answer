# e343 - BMI 計算

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=e343)

## 題目描述

題目要求計算身體質量指數 (BMI)，給定體重 (公斤) 和身高 (公尺)，並將結果四捨五入到小數點以下一位數。

## 解題思路

此題為簡單的數學計算題。根據 BMI 的公式 `BMI = 體重 / (身高 * 身高)`，直接將輸入的體重和身高代入公式計算即可。由於題目要求輸出結果四捨五入到小數點以下一位數，可以使用 `std::fixed` 和 `std::setprecision` 來格式化輸出。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
#include <iomanip>

using namespace std;

int main(){
	
	double a,b,bmi=0;
	while(cin >> a >> b){
		bmi=a/((b/100)*(b/100));
		cout << fixed << setprecision(1) << bmi << endl;
	}
}
