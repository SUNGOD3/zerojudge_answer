# f441 - 評分系統 Score

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=f441)

## 題目描述

題目要求模擬一個評分系統。給定一個包含 n 個標準答案的陣列 a，以及一個測試案例的 m 分數。接著，給定 t 個測試案例，每個測試案例包含 n 個學生的答案。對於每個測試案例，計算有多少學生的答案與標準答案完全匹配，然後將匹配的數量乘以 m，作為該測試案例的總分。

## 解題思路

這題的解題思路非常直接。對於每個測試案例，我們遍歷學生的答案，並將其與標準答案陣列中的元素逐一比較。如果學生的答案與標準答案匹配，則將匹配的數量加一。最後，將匹配的數量乘以 m，得到該測試案例的總分。程式碼使用了快速的輸入輸出方式，以提高效率。

## 複雜度分析
- 時間複雜度: O(n*t)
- 空間複雜度: O(n)

## 程式碼
```cpp
#pragma GCC optimize("Ofast,unroll-loops,no-stack-protector,fast-math")
#pragma GCC target("sse,sse2,sse3,ssse3,sse4,popcnt,abm,mmx,avx,tune=native")
#include <stdio.h>
inline int read(){
	int a(0);
	char c('0');
	while(c>='0'){
		a=(a<<3)+(a<<1)+c-'0';
		c=getchar_unlocked();
	}
	return a;
}
inline void write(int x) {
	if(x==0)
		putchar_unlocked('0');
	else{
		int stk[6],*ptr(&stk[0]);
		while(x){*ptr=x%10;x/=10;++ptr;}
		while(--ptr>=(&stk[0])){putchar_unlocked(*ptr+'0');}
	}
}
int main(){
	int n=read(),m=read(),t;
	int a[n],b;
	for(int i=0;i<n;++i)
		a[i]=read(); 
	t=read();
	while(t--){
		int ans=0;
		for(int i=0;i<n;++i){
			b=read();
			if(b==a[i])++ans;
		}
		write(ans*m);
		putchar_unlocked('\n');
	}
}
