# f046 - 公車上的跑馬燈

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=f046)

## 題目描述

題目要求模擬一個跑馬燈的顯示效果。給定跑馬燈的寬度 `w` 和要顯示的訊息 `s`，以及時間 `n`，輸出在時間 `n` 時跑馬燈上顯示的文字。跑馬燈的顯示方式是訊息從右向左滾動，訊息的長度可能大於跑馬燈的寬度。

## 解題思路

解題思路是先在訊息的左右兩側添加空格，使得訊息的長度大於跑馬燈的寬度。然後，根據時間 `n` 計算出顯示的起始位置，並從訊息中提取出對應長度的子字串進行輸出。由於跑馬燈的顯示是從右向左滾動的，因此起始位置的計算方式是 `n`。

## 複雜度分析
- 時間複雜度: O(w)
- 空間複雜度: O(n)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int w,n;
string s;
int main(){
	cin >> w;
	getline(cin,s);
	getline(cin,s);
	for(int i=0;i<w;++i){
		s=' '+s+' ';
	}
	
	cin >> n;
	for(int i=0,it=n;i<w;++i,++it){
		cout << s[it];
	}
}
