# n621 - DD說這題是簡單題

[🔗 前往 ZeroJudge 原題](https://zerojudge.tw/ShowProblem?problemid=n621)

## 題目描述
題目要求計算 `x` 除以 `y` 的商，如果 `x` 除以 `y` 有餘數，則商加 1。換句話說，計算 `ceil(x / y)`，其中 `ceil` 表示向上取整。

## 解題思路
題目可以直接使用整數除法和模運算符來解決。`x / y` 計算的是商，`x % y` 計算的是餘數。如果餘數不為 0，則表示需要將商加 1。

## 複雜度分析
- 時間複雜度: O(1)
- 空間複雜度: O(1)

## 程式碼
```cpp
#include <iostream>
using namespace std;
int main(){
	int x,y;
	cin >> x >> y;
	cout << x/y + (x%y?1:0);
}
