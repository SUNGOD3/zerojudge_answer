# 猥瑣罐頭下樓梯

## 題目描述

有一天，猥瑣罐頭在下樓梯，不知怎麼了，爬了好久都看不到盡頭..... 圖片來源 : http://znowledge.blogspot.com/2007/11/blog-post_3307.html  罐頭從地下 0 樓開始往下走( 去找猥瑣桑葉? )，他只能一次走 1 或 2 步!問 : 給你地下共有幾樓，請問罐頭有幾種走法?

## 輸入說明

每一行有一個正整數N ( 1 <= N <= 2147483647 )，代表總共有幾層

## 輸出說明

輸出罐頭有幾種走法! 歐! 對了! 因為答案可能很大， 請把答案 MOD 10007 ( MOD 代表取餘數 )
