# 11428 - Cubes

## 題目描述

給定一個正整數 N，你必須找到兩個正整數 x 和 y，使得：
N = x^3 - y^3

## 輸入說明

每行有一個整數 N (0 < N <= 10000)
當 N = 0 代表輸入結束。

## 輸出說明

對於每行 N，請輸出正整數 x 和 y 使等式 N = x^3 - y^3 成立
若無此正整數對，請輸出"No solution"
若有多組答案滿足，請輸出 y 最小的那一組
