# Bert 爬樓梯

## 題目描述

BERT 是個奇怪的人，爬樓梯都不乖乖乖的一次走一格，而是有時走一階，或一次走兩階。
換句話說，假設階梯有三階，那他有三種走法
一： 第一步走一階，第二步走二階。
二： 第一步走二階，第二步走一階。
三： 全程都走一階。
 
現在請你寫個程式，算出如果是 n  格樓梯，Bert 有幾種走法 ?

## 輸入說明

多筆輸入
每行一個數字 N ，代表現在有 N 階樓梯 。
(1 <= Ｎ <= 10000)

## 輸出說明

每行一個數字，代表當樓梯有 N 階時答案。
答案可能很大，請 mod 1000000007
