# pB. 數字拆解

## 題目描述

給定一個正整數 n，求 n 能夠拆解成幾種相異的數字組合。舉例來說，當 n = 332 + 11 + 1 + 1共三種組合，其中 2 + 1 和 1 + 2 視為同種組合。舉例來說，當 n = 443 + 12 + 22 + 1 + 11 + 1 + 1 + 1共五種組合

## 輸入說明

正整數 n （1 ≤ n ≤ 100）

## 輸出說明

n 能夠拆解的數字組合數量
